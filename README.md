## üõ† Features

### Frontend
- Single Page Application (SPA) using **React Router**  
- Responsive design with **Tailwind CSS**  
- Lazy loading and Suspense for routes  
- Professional **loading skeletons** for better UX  
- Cart system with quantity management  
- Reservation & booking system  
- Email verification & password reset flows  
- Google reCAPTCHA v2 for security  
- Toast notifications using **React Toastify**

### Backend
- **Node.js** and **Express** server  
- RESTful API endpoints  
- MongoDB database for users, orders, reservations, and products  
- JWT authentication for secure login  
- Password hashing and email verification  
- CORS configured for frontend deployment

- **Backend:** Node.js, Express, MongoDB, Mongoose, JWT, Bcrypt, Nodemailer  
- **Deployment:** Render.com for both frontend and backend
